**EduFlow** is a simple, interactive **Streamlit app** that helps students manage homework, track due dates, and stay motivated with daily quotes. 
Everything is lightweight, easy to use, and designed to make studying just a little bit more fun.  

---

## Features

- **Homework Stack**: Add, view, and complete tasks in a stack-based system.  
- **Due Date Alerts**: Tasks are color-coded by urgency (Low / Medium / High).  
- **Motivational Quotes**: Random quote every time you open the app or complete a task.  
- **User-Friendly Dashboard**: Quick overview of total tasks and top-priority assignment.  
- **Persistent Storage**: Tasks are saved locally in `homework_data.json`.  

---

## Tech Stack

- **Frontend & Backend**: Streamlit (Python)  
- **Data Handling**: JSON, datetime  
- **Python Version**: 3.10+ recommended  

---

## Installation & Running

1. **Clone the repo**  
```
git clone https://github.com/your-username/eduflow.git
cd eduflow
```

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://blank-app-template.streamlit.app/)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
